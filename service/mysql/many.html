# 多表查询

# 多表的关系

  • 概述：项目开发中，在进行数据库设计时，会根据业务需求以及业务模块之间的关系，分析并设计表结构，由于业务之间相互关联，所以表结构之间也存在着多种关系，基本分为三种：
    • 一对多(多对一)
      • 例如：部门与员工的关系，一个部门对应多个员工
      • 实现：在多的乙方建立外键，只想一的一方的主键
    • 多对多
      • 例如：学生与课程的关系，一个学生可以选择多个课程，课程包含多个学生
      • 实现：建立第三种中间变，中间表至少包含两个外键，分辨关联双方主键
    • 一对一
      • 例如：用户与用户详情的关系。一对一，多用于单表拆分，将一张表的基础字段放在一张表里，将其他详情字段放在另一张表中。
      • 实现：在任意一方加入外键，关联另一方的主键，并且设置外键为唯一(unique)

# 多表查询概述

  • 概述：从多张表中查询数据
  • 笛卡尔积：笛卡尔乘积是指在数学中，两个集合A和集合B的所有组合情况（在多表查询时，需要消除无效的笛卡尔积）
select * from user, department where user.department_id = department.id and ;

# 内连接

  • 概述：相当于查询A、B交集部分数据。

# 隐式内连接

select * from table1, table2 where table1.id = table2.tid;

# 显示内连接

-- inner join 可以缩写为join
select * from table1 join table2 on table1.id = table2.tid;

# 外连接

# 左外连接

-- left outer join 可以缩写为 left join
select * from table1 left join table2 on table1.id = table2.tid;

# 右外连接

-- right outer join 可以缩写为 right join
select * from table1 right join table2 on table1.id = table2.tid;

# 自连接

  • 自连接查询中表名称必须要起别名
    • 自连接查询，可以是内连接查询，也可以是外连接查询
-- 内连接
select * from table1 aliasname join table1 aliasnameb on aliasname.id = aliasnameb.name
-- 外连接
select * from table1 a left join table2 b on a.id = b.tid;

# 联合查询

  • 联合查询就是将多次查询的接口合并起来，形成一个新的查询结果

注意

  1. 对于联合查询多张表的列数必须保持一致，字段类型也需要保持一致
-- union all 查全部，可能会有重复。union 会对查询结果去重
select * from table1
union
select * from table2

# 子查询

  • 概述：SQL语句中嵌套select查询语句，称为嵌套查询，又称子查询
select * from table1 where column1 = (select column1 from table2)

# 标量子查询

  • 子查询结果为单个值
  • 常用操作符号：=、<、>、>=、<=
-- 查询 deparemtn.id 为 1 的所有用户信息
select * from user where department_id = (select id from department where id = 1);

# 列子查询

  • 子查询结果为一列
  • 常用操作符：
操作符 描述
in 在指定的集合范围内，多选一
not in 不在指定的集合范围之内
any 子查询返回列表中，有任意一个满足即可
some 同上，一般都可以使用any
all 子查询返回列表的所有值都必须满足
-- 
-- 查询部门最大 id下的所有人
select * from user where department_id < any (
  select max(id) from department 
)

-- 查询 department_id 小于 1和2 的
select * from user where department_id <= all (
  select id from department where id = 1 or id = 2
)

# 行子查询

  • 子查询结果为一行
select * from user where (department_id, id) = (子查询)

# 表子查询

  • 子查询结果为多行多列
select * from user where (department_id, id) in (子查询)
